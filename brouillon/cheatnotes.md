# Cheat Sheet Markdown

## Titres
```
# Titre de niveau 1
## Titre de niveau 2
### Titre de niveau 3
```

## Texte en gras et en italique
```
**Texte en gras**
*Texte en italique*
```

## Listes
### Liste non ordonnée
```
- Élément 1
- Élément 2
```
### Liste ordonnée
```
1. Élément 1
2. Élément 2
```

## Liens
```
[Texte du lien](URL)
```

## Images
```
![Texte alternatif](URL_de_l'image)
```

## Citations
```
> Ceci est une citation.
```

## Code
### Code en ligne
```
`code`
```
### Bloc de code
```
` ` `
code
` ` `
```

## Tableaux
```
| Colonne 1 | Colonne 2 |
| --------- | --------- |
| Ligne 1   | Ligne 1   |
| Ligne 2   | Ligne 2   |
```

## Séparateurs
```
---
```

## Échappement de caractères
```
\*Texte en italique\*
```
